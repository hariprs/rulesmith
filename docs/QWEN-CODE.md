# RuleSmith for Qwen Code

**Adapter Pattern Integration**

---

## Overview

Qwen Code is Alibaba's AI coding assistant. RuleSmith can integrate with Qwen Code through prompt adapters and custom instruction systems.

---

## Prerequisites

1. **Qwen Code** or **Qwen Developer** access
2. **Qwen API credentials** (if using API mode)
3. **Python 3.8+** (for adapter scripts)

---

## Installation

### Method 1: Direct Prompt Integration



## Qwen-Specific Configuration

### Custom Instructions Format

Qwen uses a custom instructions format. Create `.qwen/instructions.md`:

```markdown
# Qwen Code Custom Instructions
# Generated by RuleSmith

## Role
You are an expert coding assistant with knowledge of:
- The user's coding preferences (learned from conversation analysis)
- Project-specific conventions
- Team coding standards

## Preferences
- [Generated by /rulesmith command]

## Output Format
- Provide working code examples
- Include brief explanations
- Suggest alternatives when applicable
```

## Qwen Adapter Pattern

### Adapter Script

```python
# ~/.qwen/adapter/self-improvement.py
"""
Adapter for RuleSmith to work with Qwen Code.
"""

import json
import os
from pathlib import Path
from typing import List, Dict

class QwenSelfImprovementAdapter:
    """Adapt self-improvement patterns to Qwen Code format."""

    def __init__(self, state_path: str = None):
        self.state_path = state_path or os.path.expanduser(
            "~/.qwen/state/self-improvement.json"
        )
        self.prompts_path = os.path.expanduser(
            "~/.qwen/prompts/self-improvement/"
        )

    def load_conversation_analysis_prompt(self) -> str:
        """Load conversation analysis prompt for Qwen."""
        prompt_file = Path(self.prompts_path) / "analyze-conversation.md"
        with open(prompt_file) as f:
            content = f.read()

        # Adapt for Qwen's prompt format
        return self._adapt_for_qwen(content)

    def _adapt_for_qwen(self, prompt: str) -> str:
        """Convert Claude-style prompt to Qwen format."""
        # Qwen prefers more explicit instructions
        adapted = f"""
## Task: Analyze Conversation for Pattern Extraction

{prompt}

## Output Format
Please provide your analysis as a JSON object with the following structure:
{{
  "corrections_found": <number>,
  "patterns": [
    {{
      "pattern": "<pattern description>",
      "count": <occurrence count>,
      "examples": ["<example 1>", "<example 2>"],
      "suggested_rule": "<Qwen-compatible rule>"
    }}
  ]
}}
"""
        return adapted

    def generate_qwen_instructions(self, patterns: List[Dict]) -> str:
        """Generate Qwen custom instructions from patterns."""
        instructions = ["# Qwen Code Custom Instructions\n"]
        instructions.append("## Learned Preferences\n\n")

        for pattern in patterns:
            instructions.append(f"- {pattern['suggested_rule']}\n")

        instructions.append("\n## Communication Style\n")
        instructions.append("- Keep responses concise and actionable\n")
        instructions.append("- Provide working code examples\n")
        instructions.append("- Explain trade-offs when relevant\n")

        return "".join(instructions)

    def update_state(self, patterns: List[Dict]) -> None:
        """Update Qwen state with new patterns."""
        state = self._load_state()
        state["patterns_found"].extend(patterns)
        state["last_analysis"] = datetime.now().isoformat()
        self._save_state(state)

    def _load_state(self) -> Dict:
        """Load current state."""
        if os.path.exists(self.state_path):
            with open(self.state_path) as f:
                return json.load(f)
        return {
            "patterns_found": [],
            "last_analysis": None,
            "improvements_applied": 0
        }

    def _save_state(self, state: Dict) -> None:
        """Save state to file."""
        os.makedirs(os.path.dirname(self.state_path), exist_ok=True)
        with open(self.state_path, "w") as f:
            json.dump(state, f, indent=2)


# CLI interface
def main():
    import sys

    adapter = QwenSelfImprovementAdapter()

    if len(sys.argv) > 1 and sys.argv[1] == "--analyze":
        # Run conversation analysis
        prompt = adapter.load_conversation_analysis_prompt()
        print(prompt)
    elif len(sys.argv) > 1 and sys.argv[1] == "--stats":
        # Show statistics
        state = adapter._load_state()
        print(f"Patterns Found: {len(state['patterns_found'])}")
        print(f"Last Analysis: {state['last_analysis']}")
    else:
        print("Usage: self-improvement.py [--analyze|--stats]")


if __name__ == "__main__":
    main()
```

## Advanced Features

### Multi-Language Support

Qwen Code supports multiple programming languages. Configure per-language rules:

```markdown
## Language-Specific Rules

### Python
- Follow PEP 8 guidelines
- Use type hints for all functions
- Prefer list comprehensions over map/filter

### JavaScript/TypeScript
- Use TypeScript strict mode
- Prefer async/await over promises
- Use 'unknown' instead of 'any'

### Java
- Follow Java naming conventions
- Use Optional for nullable returns
- Prefer streams over loops
```

### Context-Aware Analysis

Configure Qwen to analyze context:

```python
# context_analyzer.py
class QwenContextAnalyzer:
    """Analyze code context for Qwen-specific patterns."""

    def analyze_file_context(self, file_path: str) -> Dict:
        """Analyze file for Qwen-relevant context."""
        return {
            "language": self._detect_language(file_path),
            "framework": self._detect_framework(file_path),
            "patterns": self._extract_patterns(file_path)
        }
```

## Troubleshooting

### Qwen API Errors

**Issue:** API authentication failures

**Solutions:**
```bash
# Check API key
echo $QWEN_API_KEY

# Re-set API key
export QWEN_API_KEY="your-new-key"

# Test connection
curl -H "Authorization: Bearer $QWEN_API_KEY" \
  https://api.qwen.ai/v1/models
```

### Adapter Script Errors

**Issue:** Python adapter not working

**Solutions:**
```bash
# Check Python version
python --version  # Should be 3.8+

# Install dependencies
pip install qwen-client

# Reinstall adapter
python ~/.qwen/adapter/self-improvement.py --install
```

### Custom Instructions Not Loading

**Issue:** Qwen not using custom instructions

**Solutions:**
1. Verify file path in Qwen settings
2. Check file is valid Markdown
3. Reload Qwen Code interface
4. Check for syntax errors in instructions

## FAQ

**Q: Does Qwen Code support slash commands?**

A: Not natively. Use the adapter script or paste prompts directly.

**Q: Can I use this with Qwen-turbo?**

A: Yes. The adapter works with all Qwen models (turbo, plus, pro).

**Q: How does this compare to Claude Code integration?**

A: Functionality is similar but requires adapter scripts since Qwen doesn't have native skill support.

**Q: Can I share rules between Qwen and other platforms?**

A: Yes. The core patterns are platform-agnostic. Only the output format differs.
