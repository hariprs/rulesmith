# Project-Self-Improvement for OpenCLAW

**Custom Instructions Integration**

---

## Overview

OpenCLAW is an AI-powered coding assistant. Project-Self-Improvement integrates with OpenCLAW through its custom instructions and prompt configuration system.

---

## Prerequisites

1. **OpenCLAW** desktop app or browser access
2. **OpenCLAW API key** (if using API mode)
3. **Basic familiarity** with OpenCLAW's configuration

---

## Installation

### Method 1: OpenCLAW Desktop App

```bash
# Clone repository
git clone https://github.com/yourusername/Project-Self_Improvement.git
cd Project-Self_Improvement

# Copy prompts to OpenCLAW configuration directory
mkdir -p ~/.openclaw/prompts/
cp -r .claude/skills/rulesmith/prompts/ ~/.openclaw/prompts/self-improvement/

# Copy state management files
mkdir -p ~/.openclaw/data/
cp -r .claude/skills/rulesmith/data/ ~/.openclaw/data/self-improvement/
```

### Method 2: Browser Extension

```bash
# For OpenCLAW browser extension
mkdir -p ~/openclaw-extension/skills/
cp -r .claude/skills/rulesmith/ ~/openclaw-extension/skills/

# Enable in extension settings
# Settings → Advanced → Custom Skills Path: ~/openclaw-extension/skills/
```

### Method 3: Project-Level Integration

```bash
# Within your project directory
mkdir -p .openclaw/
cp -r .claude/skills/rulesmith/prompts/ .openclaw/prompts/
echo "{extends: ['.openclaw/prompts/self-improvement/']}" > .openclaw/config.json
```

## OpenCLAW Configuration

### Custom Instructions File

Create `~/.openclaw/instructions.md`:

```markdown
# OpenCLAW Custom Instructions
# Managed by Project-Self-Improvement

## Assistant Role
You are an expert coding assistant that learns from user feedback.

## Learned Preferences
<!-- Generated by /improve-rules command -->
<!-- Last updated: [timestamp] -->

## Communication Guidelines
- Be concise and direct
- Provide working code examples
- Explain trade-offs when relevant
- Ask clarifying questions when requirements are ambiguous

## Output Format
- Use code blocks for all code
- Include brief explanations
- Highlight important considerations
```

### OpenCLAW Config File

Create `~/.openclaw/config.json`:

```json
{
  "version": "1.0",
  "skills": {
    "enabled": ["self-improvement"],
    "paths": [
      "~/.openclaw/prompts/self-improvement/"
    ]
  },
  "state": {
    "file": "~/.openclaw/data/self-improvement/state.json",
    "backup_dir": "~/.openclaw/data/self-improvement/history/"
  },
  "commands": {
    "improve_rules": {
      "trigger": ["/improve-rules", "/analyze", "/learn"],
      "action": "analyze_conversation"
    },
    "show_stats": {
      "trigger": ["/improve-rules --stats"],
      "action": "show_statistics"
    }
  }
}
```

In OpenCLAW chat interface:


### Alternative Triggers

OpenCLAW supports multiple trigger patterns:

| Trigger | Action |
|---------|--------|
| `/improve-rules` | Full conversation analysis |
| `/analyze` | Quick pattern detection |
| `/learn` | Learn from current conversation |
| `/improve-rules --stats` | Show performance metrics |
| `/improve-rules --history` | Show recent improvements |

## OpenCLAW API Integration

### Direct API Usage

```python
# openclaw_self_improvement.py
import os
import requests
import json

class OpenCLAWSelfImprovement:
    """Integrate self-improvement with OpenCLAW API."""

    def __init__(self):
        self.api_key = os.getenv("OPENCLAW_API_KEY")
        self.base_url = "https://api.openclaw.ai/v1"
        self.state_file = os.path.expanduser("~/.openclaw/data/self-improvement/state.json")

    def analyze_conversation(self, conversation_history: list) -> dict:
        """Analyze conversation for patterns."""
        # Load analysis prompt
        prompt = self._load_analysis_prompt()

        # Call OpenCLAW API
        response = requests.post(
            f"{self.base_url}/chat/completions",
            headers={"Authorization": f"Bearer {self.api_key}"},
            json={
                "model": "openclaw-coder-v2",
                "messages": [
                    {"role": "system", "content": prompt},
                    {"role": "user", "content": f"Analyze this conversation:\n{conversation_history}"}
                ]
            }
        )

        return response.json()

    def update_instructions(self, patterns: list) -> None:
        """Update OpenCLAW instructions with new patterns."""
        instructions = self._generate_instructions(patterns)

        # Save to instructions file
        with open(os.path.expanduser("~/.openclaw/instructions.md"), "w") as f:
            f.write(instructions)

    def _load_analysis_prompt(self) -> str:
        """Load conversation analysis prompt."""
        prompt_file = os.path.expanduser("~/.openclaw/prompts/self-improvement/analyze-conversation.md")
        with open(prompt_file) as f:
            return f.read()

    def _generate_instructions(self, patterns: list) -> str:
        """Generate OpenCLAW instructions from patterns."""
        instructions = ["# OpenCLAW Custom Instructions\n\n"]
        instructions.append("## Learned Preferences\n\n")

        for pattern in patterns:
            instructions.append(f"- {pattern['suggested_rule']}\n")

        return "".join(instructions)
```

## Troubleshooting

### Instructions Not Loading

**Issue:** OpenCLAW not using custom instructions

**Solutions:**
1. Verify file path in OpenCLAW settings
2. Check file is readable (`chmod 644 ~/.openclaw/instructions.md`)
3. Reload OpenCLAW (Ctrl+R or restart app)
4. Check for Markdown syntax errors

### Command Not Recognized

**Issue:** `/improve-rules` command not working

**Solutions:**
```bash
# Verify prompt files exist
ls -la ~/.openclaw/prompts/self-improvement/

# Check config.json syntax
cat ~/.openclaw/config.json | python -m json.tool

# Reinstall prompts
cp -r /path/to/Project-Self_Improvement/.claude/skills/rulesmith/prompts/ \
      ~/.openclaw/prompts/self-improvement/
```

### State File Errors

**Issue:** State file corrupted or missing

**Solutions:**
```bash
# Reset state file
cat > ~/.openclaw/data/self-improvement/state.json << 'EOF'
{
  "last_analysis": null,
  "patterns_found": [],
  "improvements_applied": 0,
  "corrections_reduction": 0,
  "platform": "openclaw"
}
EOF

# Or restore from backup
cp ~/.openclaw/data/self-improvement/history/rules-<latest>.md \
   ~/.openclaw/data/self-improvement/state.json
```

## FAQ

**Q: Can I use OpenCLAW with multiple projects?**

A: Yes. Use `.openclaw/` directory in each project for project-specific rules.

**Q: How do I reset all learned patterns?**

A: Delete `~/.openclaw/data/self-improvement/state.json` and restart OpenCLAW.

**Q: Can I export/import rules?**

A: Yes. Copy `~/.openclaw/instructions.md` to share rules across machines.

**Q: Does this work with OpenCLAW Pro?**

A: Yes. Works with all OpenCLAW tiers (Free, Pro, Enterprise).
